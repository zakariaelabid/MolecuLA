# Curated Results

This folder contains compact artifacts copied from the source study:

- JSON summaries
- compact CSV metrics and probe tables
- selected PNG figures
- HTML latent visualizations

Excluded intentionally:

- full RDKit/property panels
- latent arrays
- `.npy` and `.npz` caches
- duplicate 300-550 MB CSVs
- regenerated MLP weight files

The excluded files are reproducible from the notebooks and are too large or too redundant for a GitHub-ready repository.
